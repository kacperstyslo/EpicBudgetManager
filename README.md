# EpicBudgetManager

## Description
A small project to manage budget that helped me to get the basic knowledge of tkinter, numpy and pandas modules

## Requirements
Python 3.5+

## Start-up
```commandline
On Windows:
python manager.py
On Linux:
python3 manager.py
```
## Tools Used
* Python
* Pandas
* Numpy
